## ⚡ Команды запуска

### Установка зависимостей
```bash
npm run install:all
```

### Запуск в режиме разработки
```bash
npm run dev
# или через скрипт:
scripts/dev-start.bat
```

### Отдельный запуск сервисов
```bash
npm run dev:backend    # Backend на порту 3000
npm run dev:frontend   # Frontend на порту 5000
```

### Сборка для продакшна
```bash
npm run build
```

### Очистка проекта
```bash
npm run clean
```
